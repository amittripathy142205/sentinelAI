# SmartSentinel AI Backend

AI-powered home security surveillance API with:
- 🔥 Fire & Smoke Detection
- 👤 Face Recognition
- 🚪 Door Monitoring
- 📊 Real-time Analytics

## API Endpoints

- `POST /login` - User authentication
- `POST /register` - User registration
- `GET /stats` - Dashboard statistics
- `GET /incidents` - Incident list
- `GET /evidence/*` - Evidence files
